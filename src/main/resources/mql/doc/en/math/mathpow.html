MathPow

The function raises a base to a specified power.

double  MathPow(
   double  base,         // base 
   double  exponent      // exponent value
   );

Parameters

base

[in]  Base.

exponent

[in]  Exponent value.

Return Value

Value of base raised to the specified power.

Note

Instead of MathPow() you can use pow().
