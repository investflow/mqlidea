SetLevelValue

The function sets a value for a given horizontal level of the indicator to be output in a separate window.

void  SetLevelValue(
   int     level,       // level
   double  value        // value
   );

Parameters

level

[in]  Level index (0-31).

value

[in]  Value for the given indicator level.

Returned value

None.

Example:

   SetLevelValue(1,3.14);
