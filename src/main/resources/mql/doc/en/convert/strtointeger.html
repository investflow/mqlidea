StrToInteger

Converts string containing the value character representation into a value of the int (integer) type.

int  StrToInteger(
   string  value      // string
   );

Parameters

value

[in]  String containing the integer character representation format.

Returned value

Value of int type.

Example:

  int var1=StrToInteger("1024");
