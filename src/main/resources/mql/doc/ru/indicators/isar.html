iSAR

Возвращает значение индикатора Parabolic Stop and Reverse system.

double  iSAR(
   string       symbol,            // имя символа
   int          timeframe,         // таймфрейм
   double       step,              // шаг изменения цены - коэффициент ускорения
   double       maximum,           // максимальный шаг
   int          shift              // сдвиг
   );

Параметры

symbol

[in]  Символьное имя инструмента, на данных которого будет вычисляться индикатор. NULL означает текущий символ.

timeframe

[in]  Период. Может быть одним из значений перечисления ENUM_TIMEFRAMES. 0 означает период текущего графика.

step

[in]  Шаг изменения цены, обычно 0.02.

maximum

[in]  Максимальный шаг, обычно 0.2.

shift

[in]  Индекс получаемого значения из индикаторного буфера (сдвиг относительно текущего бара на указанное количество периодов назад).

Возвращаемое значение

Значение технического индикатора Parabolic Stop and Reverse system.

Пример:

  if(iSAR(NULL,0,0.02,0.2,0)>Close[0]) return(0);
