ArrayDimension

Возвращает ранг многомерного массива.

int  ArrayDimension(
   void&  array[]           // массив
   );

Параметры

array[]

[in]  Массив, для которого будет возвращен ранг.

Возвращаемое значение

Размерность многомерного массива, либо -1 в случае неудачи.

Пример:

void OnStart()
  {
   int num_array[10][5];
   int dim_size=ArrayDimension(num_array);// dim_size=2
   Print("Размерность массива num_array=",dim_size); 
  }
