StringGetChar

Возвращает значение символа, расположенного в указанной позиции строки.

int  StringGetChar(
   string  string_value,     // строка
   int     pos               // позиция символа в строке
   );

Параметры

string_value

[in]  Строка.

pos

[in]  Позиция символа в строке. Может быть от 0 до StringLen(text) -1.

Возвращаемое значение

Код символа, либо при 0 в случае ошибки. Для получения кода ошибки нужно вызвать функцию GetLastError().

Пример:

  int char_code=StringGetChar("abcdefgh", 2);
  // символьный код 'c' = 99
