StringTrimRight

Удаляет символы перевода каретки, пробелы и символы табуляции в правой части строки.

string  StringTrimRight(
   const string  text      // строка
   );

Параметры

text

[in]  Строка, которая должна быть обрезана справа.

Возвращаемое значение

Копия преобразованной строки, если это возможно, в противном случае возвращается пустая строка.

Пример:

  string str1="  Hello world   ";
  string str2=StringTrimRight(str1);
  // после срезания у переменной str2 будет значение - "  Hello World"
