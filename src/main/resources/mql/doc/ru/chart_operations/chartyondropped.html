ChartYOnDropped

Возвращает координату по оси Y, соответствующую точке, в которой брошен мышкой данный эксперт или скрипт.

int  ChartYOnDropped();

Возвращаемое значение

Значение координаты Y.

Примечание

Ось Y направлена сверху вниз.
