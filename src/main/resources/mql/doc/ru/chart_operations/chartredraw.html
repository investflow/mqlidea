ChartRedraw

Вызывает принудительную перерисовку указанного графика.

void  ChartRedraw(
   long  chart_id=0      // идентификатор графика
   );

Параметры

chart_id=0

[in]  Идентификатор графика. 0 означает текущий график.

Примечание

Обычно применяется после изменения свойств объектов.

При вызове функции ChartRedraw() из индикатора график будет перерисован только после окончания расчетов в функции OnCalculate(), так как индикаторы выполняются в общем потоке выполнения терминала.
