WindowIsVisible

Возвращает состояние видимости подокна графика.

int  WindowIsVisible(
   int       index     // индекс подокна графика
   );

Параметры

index

[in]  Индекс подокна графика.

Возвращаемое значение

Состояние видимости подокна графика (true, если подокно графика видимо, иначе false). Подокно графика может быть скрыто из-за свойств видимости помещенного в него индикатора.

Пример:

  int maywin=WindowFind("MyMACD");
  if(maywin>-1 && WindowIsVisible(maywin)==true)
    Print("окно MyMACD видимое");
  else
    Print("окно MyMACD не обнаружено или невидимое");
