AccountFreeMargin

Возвращает значение свободных средств, разрешенных для открытия ордеров на текущем счете.

double  AccountFreeMargin();

Возвращаемое значение

Значение свободных средств, разрешенных для открытия ордеров на текущем счете.

Пример:

  Print("Свободная маржа счета = ",AccountFreeMargin());
