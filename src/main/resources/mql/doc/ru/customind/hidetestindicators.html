HideTestIndicators

Выставляет флаг скрытия индикаторов, вызываемых экспертом.

void  HideTestIndicators(
   bool      hide     // флаг
   );

Параметры

hide

[in]  Флаг (true - если нужно прятать индикаторы, иначе false).

Возвращаемое значение

Нет.

Примечание

При открытии графика после тестирования индикаторы, помеченные флагом скрытия, не будут выведены на график тестирования. Перед каждым вызовом индикатор помечается текущим установленным флагом скрытия. На график тестирования могут быть выведены только те индикаторы, которые непосредственно вызываются из тестируемого эксперта.

Пример:

   HideTestIndicators(true);
   MaCurrent=iMA(NULL,0,56,0,MODE_EMA,PRICE_CLOSE,0);
   MaPrevious=iMA(NULL,0,56,0,MODE_EMA,PRICE_CLOSE,1);
   HideTestIndicators(false);
