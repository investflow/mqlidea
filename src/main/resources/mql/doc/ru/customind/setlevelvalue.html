SetLevelValue

Устанавливает значение указанного горизонтального уровня индикатора, выводимого в отдельное окно.

void  SetLevelValue(
   int     level,       // номер уровня
   double  value        // значение
   );

Параметры

level

[in]  Номер уровня (0-31).

value

[in]  Значение для указанного уровня.

Возвращаемое значение

Нет.

Пример:

   SetLevelValue(1,3.14);
