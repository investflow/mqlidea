SetLevelStyle

Устанавливает новый стиль, ширину и цвет для горизонтальных уровней индикатора, выводимого в отдельное окно.

void  SetLevelStyle(
   int     draw_style,       // стиль линии
   int     line_width,       // ширина
   color   clr               // цвет
   );

Параметры

draw_style

[in]  Стиль линии. Может быть одним из перечисленных стилей линии. Пустое значение (EMPTY) указывает, что стиль не будет изменен.

line_width

[in]  Ширина линии. Допустимые значения - 1,2,3,4,5. Пустое значение (EMPTY) указывает, что ширина не будет изменена.

clr

[in]  Цвет линии. Пустое значение CLR_NONE указывает, что цвет не будет изменен.

Возвращаемое значение

Нет.

Пример:

//--- уровни в виде толстой красной линии
   SetLevelStyle(STYLE_SOLID,2,clrRed);
