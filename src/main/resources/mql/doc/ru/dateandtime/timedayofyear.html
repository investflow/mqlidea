TimeDayOfYear

Возвращает день года указанной даты.

int  TimeDayOfYear(
   datetime      date            // дата и время
   );

Параметры

date

[in]  Дата, представленная в виде количества секунд, прошедших после 00:00 1 января 1970 года.

Возвращаемое значение

День года (1 - 1 января,..,365(6) - 31 декабря) указанной даты.

Пример:

  int nday=TimeDayOfYear(TimeCurrent());
