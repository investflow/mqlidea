TimeSeconds

Возвращает количество секунд, прошедших с начала минуты указанного времени.

int  TimeSeconds(
   datetime      date            // дата и время
   );

Параметры

date

[in]  Дата, представленная в виде количества секунд, прошедших после 00:00 1 января 1970 года.

Возвращаемое значение

Количество секунд, прошедших с начала минуты указанного времени.

Пример:

  int m=TimeSeconds(TimeCurrent());
