DebugBreak

Программная точка останова при отладке.

void  DebugBreak();

Возвращаемое значение

Нет возвращаемого значения.

Примечание

Прерывание выполнения mql4-программы происходит только в том случае, если программа запущена в режиме отладки. Функцию можно использовать для просмотра значений переменных и/или дальнейшего пошагового выполнения.
