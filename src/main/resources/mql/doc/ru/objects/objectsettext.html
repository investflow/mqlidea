ObjectSetText

Изменяет описание объекта.

bool  ObjectSetText(
   string  object_name,         // имя объекта
   string  text,                // текст описания
   int     font_size=0,         // размер шрифта
   string  font_name=NULL,      // наименование шрифта
   color   text_color=clrNONE   // цвет текста
   );

Параметры

object_name

[in]  Имя объекта.

text

[in]  Текст описания.

font_size=0

[in]  Размер шрифта в пунктах.

font_name=NULL

[in]  Наименование шрифта.

text_color=clrNONE

[in]  Цвет текста.

Возвращаемое значение

Функция возвращает true в случае удачи, иначе false. Чтобы получить дополнительную информацию об ошибке, необходимо вызвать функцию GetLastError().

Примечание

Для объектов OBJ_TEXT и OBJ_LABEL описание объектов отображается на графике в виде текстовой строки. Параметры font_size, font_name и text_color используются только для объектов OBJ_TEXT и OBJ_LABEL. Для объектов других типов эти параметры игнорируются.

Пример:

  ObjectSetText("text_object","Hello world!",10,"Times New Roman",Green);
