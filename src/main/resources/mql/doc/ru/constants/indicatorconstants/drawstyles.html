Существуют 5 стилей, которыми может рисоваться линия в пользовательском индикаторе. Они применимы только при толщине линии 0 или 1.

ENUM_LINE_STYLE

Константа

Значение

Описание

0

Сплошная линия

2

Пунктирная линия

STYLE_DASHDOT

3

Штрих-пунктирная линия

STYLE_DASHDOTDOT

4

Штрих-пунктирная линия с двойными точками

Стиль линии может также использоваться для получения или установки свойства OBJPROP_STYLE объекта.

При создании пользовательского индикатора можно указать один из 6 типов рисования (способа отображения на главном окне графика или в подокне графика), значения которых указаны в списке стилей рисования.

В одном пользовательском индикаторе допустимо использовать любые виды рисования индикаторов. Каждый вид построения требует указания от одного до двух глобальных массивов для хранения данных, необходимых для рисования.
