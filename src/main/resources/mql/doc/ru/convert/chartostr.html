CharToStr

Преобразует код символа в односимвольную строку.

string  CharToStr(
   uchar  char_code      // ASCII-код символа
   );

Параметры

char_code

[in]  ASCII-код символа.

Возвращаемое значение

Строка.

Пример:

 string str="WORL"+CharToStr(68); // 68 - код для 'D'
  // результирующая строка будет WORLD
