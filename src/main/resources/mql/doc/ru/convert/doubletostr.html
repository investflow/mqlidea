DoubleToStr

Преобразует числовое значение в текстовую строку, содержащую символьное представление числа в указанном формате точности.

string  DoubleToStr(
   double  value,     // величина
   int     digits     // формат точности
   );

Параметры

value

[in]  Величина с плавающей точкой.

digits

[in]  Формат точности, число цифр после десятичной точки (0-8).

Возвращаемое значение

Строка.

Пример:

  string value=DoubleToStr(1.28473418, 5);
  // содержимое строки value - "1.28473"
