StrToDouble

Преобразует строку, содержащую символьное представление числа, в число типа double (формат двойной точности с плавающей точкой).

double  StrToDouble(
   string  value      // строка
   );

Параметры

value

[in]  Строка, содержащая символьное представление числа.

Возвращаемое значение

Значение типа double.

Пример:

  double var=StrToDouble("103.2812");
