IsLibrariesAllowed

Возвращает информацию о возможности вызова библиотек.

bool  IsLibrariesAllowed();

Возвращаемое значение

Возвращает true, если эксперт может вызвать библиотечную функцию, иначе возвращает false.

Пример:

#import "somelibrary.ex4"
  int somefunc();
  ...
  ...
  if(IsLibrariesAllowed()==false)
    {
     Print("Библиотечные вызовы запрещены.");
     return(0);
    }
  // Тело Эксперта, вызывающее внешние функции DLL
  somefunc();
