IsConnected

Возвращает состояние главного соединения клиентского терминала с сервером, по которому производится подкачка данных.

bool  IsConnected();

Возвращаемое значение

Возвращает true, если связь с сервером установлена, или false, если связь с сервером отсутствует или прервана.

Пример:

  if(!IsConnected())
    {
     Print("Связь отсутствует!");
     return(0);
    }
  // Тело скрипта, нуждающегося в открытом подключении
  // ...
