TerminalInfoString

Функция возвращает значение соответствующего свойства окружения mql4-программы. Свойство должно быть типа string.

string  TerminalInfoString(
   int  property_id      // идентификатор свойства
   );

Параметры

property_id

[in]  Идентификатор свойства. Может быть одним из значений перечисления ENUM_TERMINAL_INFO_STRING.

Возвращаемое значение

Значение типа string.
