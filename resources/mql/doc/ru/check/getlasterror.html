GetLastError

Возвращает содержимое системной переменной _LastError.

int  GetLastError();

Возвращаемое значение

Возвращает значение последней ошибки, произошедшей во время исполнения mql4-программы.

Примечание

После вызова функции содержимое переменной _LastError обнуляется.

Коды торговых ошибок для удобства дополнительно приведены отдельным списком в разделе Коды возврата торгового сервера.
