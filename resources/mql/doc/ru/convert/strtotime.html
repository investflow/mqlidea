StrToTime

Преобразует строку, содержащую время и/или дату в формате "yyyy.mm.dd [hh:mi]", в число типа datetime (количество секунд, прошедших с 01.01.1970).

datetime  StrToTime(
   string  value      // строка-дата
   );

Параметры

value

[in]  Строка в формате "yyyy.mm.dd hh:mi ".

Возвращаемое значение

Значение типа datetime, содержащее количество секунд, прошедших с 01.01.1970.

Пример:

  datetime var1,var2,var3;
  var1=StrToTime("2003.8.12 17:35");
  var2=StrToTime("17:35");      // возврат текущей даты с указанным временем
  var3=StrToTime("2003.8.12");  // возврат даты с полуночным временем "00:00"
