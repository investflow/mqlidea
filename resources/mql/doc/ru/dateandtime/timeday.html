TimeDay

Возвращает день месяца (1 - 31) указанной даты.

int  TimeDay(
   datetime      date            // дата и время
   );

Параметры

date

[in]  Дата, представленная в виде количества секунд, прошедших после 00:00 1 января 1970 года.

Возвращаемое значение

День месяца (1-31) указанной даты.

Пример:

  int day=TimeDay(D'2003.12.31');
  // день 31
