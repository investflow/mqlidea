TimeHour

Возвращает час указанного времени.

int  TimeHour(
   datetime      date            // дата и время
   );

Параметры

date

[in]  Дата, представленная в виде количества секунд, прошедших после 00:00 1 января 1970 года.

Возвращаемое значение

Час (0-23) указанной даты.

Пример:

  int h=TimeHour(TimeCurrent());
