SetIndexStyle

Устанавливает новый тип, стиль, ширину и цвет для указанной линии индикатора.

void  SetIndexStyle(
   int     index,       // номер линии
   int     type,        // тип
   int     style=EMPTY, // стиль линии
   int     width=EMPTY, // ширина линии
   color   clr=clrNONE  // цвет
   );

Параметры

index

[in]  Порядковый номер линии. Должен быть от 0 до 7.

type

[in]  Стиль отрисовки линии индикатора. Может быть одним из перечисленных стилей отрисовки линии.

style=EMPTY

[in]  Стиль линии. Используется для линий толщиной в 1 пиксель. Может быть одним из перечисленных стилей линии. Пустое значение (EMPTY) указывает, что стиль не будет изменен.

width=EMPTY

[in]  Ширина линии. Допустимые значения - 1,2,3,4,5. Пустое значение (EMPTY) указывает, что ширина не будет изменена.

clr=clrNONE

[in]  Цвет линии. Отсутствие параметра означает, что цвет не будет изменен.

Возвращаемое значение

Нет.

Пример:

  SetIndexStyle(3,DRAW_LINE,EMPTY,2,clrRed);
