IndicatorDigits

Устанавливает формат точности (количество знаков после десятичной точки) для визуализации значений индикатора.

void  IndicatorDigits(
   int     digits         // количество знаков
   );

Параметры

digits

[in]  Формат точности, число цифр после десятичной точки.

Возвращаемое значение

Нет.

Примечание

По умолчанию используется точность цены финансового инструмента, к графику которого присоединен индикатор.

Пример:

int init()
  {
//---- 2 дополнительных буфера используются для подсчета.
   IndicatorBuffers(3);
//---- настройка параметров отрисовки
   SetIndexStyle(0,DRAW_HISTOGRAM,STYLE_SOLID,3);
   SetIndexDrawBegin(0,SignalSMA);
   IndicatorDigits(Digits+2);
//---- 3 распределенных буферов индикатора
   SetIndexBuffer(0,ind_buffer1);
   SetIndexBuffer(1,ind_buffer2);
   SetIndexBuffer(2,ind_buffer3);
//---- "короткое имя" для DataWindow и подокна индикатора
   IndicatorShortName("OsMA("+FastEMA+","+SlowEMA+","+SignalSMA+")");
//---- инициализация сделана
   return(0);
  }
