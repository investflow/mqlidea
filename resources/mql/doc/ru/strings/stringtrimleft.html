StringTrimLeft

Удаляет символы перевода каретки, пробелы и символы табуляции с начала строки до первого значимого символа.

string  StringTrimLeft(
   const string  text      // строка
   );

Параметры

text

[in]  Строка, которая должна быть обрезана слева.

Возвращаемое значение

Копия преобразованной строки, если это возможно, в противном случае возвращается пустая строка.

Пример:

  string str1="  Hello world   ";
  string str2=StringTrimLeft(str1);
  // после срезания у переменной str2 будет значение - "Hello World   "
