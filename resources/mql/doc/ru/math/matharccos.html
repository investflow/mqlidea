MathArccos

Возвращает значение арккосинуса x в диапазоне 0 к π в радианах.

double  MathArccos(
   double  val     // -1<val<1
   );

Параметры

val

[in]  Значение val между -1 и 1, арккосинус которого должен быть вычислен.

Возвращаемое значение

Арккосинус числа в радианах. Если val меньше -1 или больше 1, функция возвращает NaN (неопределенное значение).

Примечание

Вместо функции MathArccos() можно использовать функцию acos().
