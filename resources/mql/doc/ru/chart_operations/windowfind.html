WindowFind

Возвращает номер подокна графика, содержащего индикатор с указанным именем.

int  WindowFind(
   string       name   // имя
   );

Параметры

name

[in]  Короткое имя индикатора.

Возвращаемое значение

Номер подокна графика, содержащего индикатор с указанным именем name, если он найден, иначе возвращается -1.

Примечание

WindowFind() возвращает -1, если пользовательский индикатор ищет сам себя в процессе инициализации init().

Пример:

  int win_idx=WindowFind("MACD(12,26,9)");
